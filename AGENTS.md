# Agent Instructions

This file contains universal instructions for all AI agents working in this repository.

## Core Principles

1. **Write clean, maintainable code**
2. **Follow existing conventions** in the codebase
3. **Document your changes** with clear comments
4. **Test your code** before submitting

## Workflow

1. Understand the task by reading relevant files
2. Make minimal, focused changes
3. Verify the solution works
4. Update documentation if needed

## Communication

- Be concise and direct
- Ask questions when requirements are unclear
- Explain your reasoning for major decisions

## Code Standards

- Follow PEP 8 for Python code
- Use meaningful variable names
- Keep functions focused and small
- Add type hints where appropriate
